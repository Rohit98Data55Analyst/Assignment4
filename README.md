# MERN Stack Web Development – Assignment 4

## 📖 About the Project

This project is my **fourth assignment** in the **MERN Stack Web Development** course.

The objective of this assignment is to learn and practice **CSS Gradients** by creating different gradient backgrounds using pure CSS. It demonstrates how gradients can be used to design attractive and modern user interfaces without using images.

---

## 🚀 Features

* Linear Gradient
* Multiple Color Combinations
* Responsive Layout
* Clean and Simple Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
